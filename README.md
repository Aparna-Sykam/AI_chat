# Installation & Setup
  1. Clone the repository
    git clone https://github.com/Aparna-Sykam/AI_chat.git
    cd ai_chat
  2. Install dependencies
    npm install
  3. Run the application
    npm run dev
  4. Open in browser
    http://localhost:5173
